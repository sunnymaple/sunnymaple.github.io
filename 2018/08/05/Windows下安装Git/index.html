Windows下安装Git

  • Git
  • 2018-08-05

1、git下载

从官网下载：

https://git-scm.com/download/win

目前最新版本是2.220。

由于服务器是在国外，如果大家觉得下载慢，或者下载失败的话，大家可以复制链接地址：

https://github.com/git-for-windows/git/releases/download/v2.22.0.windows.1/Git-2.22.0-64-bit.exe

然后粘贴到迅雷等下载工具中下载。

2、Git安装步骤

下载好后，接下来就是安装了，双击安装文件：

点击next

Additianal icons –> On the Desktop 选中后会在桌面安装git的图标；
Windows Expoer … 选中后，鼠标右击弹出选项菜单会出现Git Bash Here和Git GUI Here；
Git LFS… 选中后支持大文件
…
本页保持默认，点击Next

选择编辑器，由于Linux是上习惯使用的是Vim编辑器，所以建议默认选择Vim编辑器，点击Next

Use Git form Git Bash only 只在git Bash中使用Git命令，不改变任何操作系统的有关环境变量，所以该选项是最安全的，所以选择该项 点击Next

选择本地库和远程库的连接方式，默认选择Use the OpenSSL library 点击Next

选择文本行末换行风格，由于Windows和Linux文件行末换行风格不一致，所以在上传（push）或者拉取（pull）文件时，需要对文件换行符进行转换，默认选择第一项即可，我们不需要过多的考虑，点击Next

选择终端工具，
Use MinTTY git默认的工具
Use window’s … 使用Window的默认的控制台窗口，它使用的是cmd命令行，
我们都知道git大部分命令和Linux命令相符合，所以建议选择第一个，如果选择第二个将和Windows命令混合使用，必然会晕，所以默认选择第一个，点击Next

默认，点击Next

git使用实验性的内置交互式添加（“git add -i”或“git add -p”）。这使得它更快（特别是启动！）但它还没有被认为是健壮的，显然这是两个新的命令，还处于试验阶段，想尝鲜的话，可以选择它，然后点击next

最后点击安装即可Install 安装即可。安装完成后，在电脑任意位置，右击，如果出现git bash here和git gui here说明安装成功。
