idea隐藏.iml/.idea/.class等文件(夹)

  • Intellij idea
  • 2019-06-27

在使用Intellij idea时，提交代码到svn或者git时，总会出现很多.iml、.xml以及target包中的class文件，而这些文件都不需要提交到svn或者git中，甚至.iml文件提交后会造成其他开发人员的不便。

那如何屏蔽这些文件呢？
首先File –> Editor –>file Types：

将Ignore files and folders输入框内的内容替换成下面的内容：

.hprof;.imi;.iml;.pyc;.pyo;.rbc;.yarb;~;.DS_Store;.git;.hg;.idea;.svn;.target;CVS;pycache;_svn;target;vssver.scc;vssver2.scc;

点击OK，然后我们在打开svn或者git的提交按钮：

这下变得很干净
